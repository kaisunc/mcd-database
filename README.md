mcd_database

Internal database for all mcd assets

1. Uploading assets
2. Tagging assets
3. Search and Filter assets
4. Manage assets

Notable components used in this project
1. Datatables(js)
2. Flask(py)
3. Flask SQLAlchemy(py, sql)
4. Postgres(db)
5. Flask Whoosh Alchemy(py)
6. Dropzone(js)
7. Selectize(js)
9. Flask Socketio(py, js)
